# Oxidation-Reduction Potential Instrument Calibration Engine Usage
## Introduction
The Oxidation-Reduction Potential Instrument Calibration Engine is a tool designed to calibrate redox instruments. This document provides a step-by-step guide on how to use the engine.
## Prerequisites
* Redox instrument with calibration capabilities
* Computer with internet connection
* Engine software installed
## Calibration Procedure
1. **Connect the instrument**: Connect the redox instrument to the computer using the provided cable.
2. **Launch the engine**: Launch the Oxidation-Reduction Potential Instrument Calibration Engine software.
3. **Select the instrument**: Select the connected redox instrument from the list of available devices.
4. **Choose the calibration method**: Choose the calibration method (e.g., single-point, multi-point) and follow the on-screen instructions.
5. **Perform the calibration**: Perform the calibration procedure according to the chosen method.
6. **Verify the results**: Verify the calibration results and adjust as necessary.
